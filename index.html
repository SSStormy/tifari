Tifari

An in-browser, taggable imageboard for artist reference images.

Self-hosted, fully open source.

Setup

Getting tifari

You can find both Windows and Linux downloads on the GitHub releases page here.

Once you have that downloaded, all you need to do is place the files somewhere that's easy to access and run tifari.exe

tifari.exe is in charge of serving the website, files and managing all the tags. Launching it will open a command shell. As long as that is open, tifari is running and can be accessed by visiting "localhost:3555" on a modern browser.

First time setup

On your first launch, you'll have to point tifari to an image folder. This folder should contain images that you want to tag using tifari.
